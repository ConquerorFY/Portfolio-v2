# Ryan's Portfolio v2

A modern, interactive 3D portfolio website built with React, Three.js, and Tailwind CSS.

## 🚀 Live Demo

Check out the live site: [ryan-lim-portfolio.netlify.app](https://ryan-lim-portfolio.netlify.app)

## ✨ Features

- **3D Interactive Models:** Immersive experience using React Three Fiber and Drei.
- **Responsive Design:** Fully optimized for all screen sizes using Tailwind CSS.
- **Dark Mode:** Support for dark and light themes.
- **Interactive Islands:** Explore different sections through a 3D island navigation.
- **Contact Form:** Functional contact form powered by EmailJS.
- **Smooth Animations:** Integrated with React Spring and Framer Motion (if applicable).

## 🛠️ Tech Stack

- **Frontend:** React.js, Tailwind CSS
- **3D Graphics:** Three.js, React Three Fiber, React Three Drei
- **Build Tool:** Vite
- **Other Tools:** Prettier (for code formatting), ESLint (for linting)

## 📦 Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/portfolio-v2.git
   ```
2. Navigate to the project directory:
   ```bash
   cd portfolio-v2
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Running Locally

To start the development server:

```bash
npm run dev
```

The application will be available at `http://localhost:5173`.

### Code Formatting

This project uses Prettier for consistent code styling. To format the entire project, run:

```bash
npm run format
```

### Building for Production

To create a production build:

```bash
npm run build
```
